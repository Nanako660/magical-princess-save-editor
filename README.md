# Magical Princess 存档编辑器

一个用于编辑《Magical Princess》游戏存档的Web工具。

## 使用方法

### 1. 获取存档文件

存档文件位置：
```
%USERPROFILE%/AppData/LocalLow/[公司名]/Magical Princess/
```

存档文件格式：
- `v10_userdata1.dat` ~ `v10_userdata31.dat` (用户存档)
- `v10_indexdata.dat` (存档索引)

### 2. 使用编辑器

1. 双击打开 `dist/index.html` 文件（需要现代浏览器）
2. 点击「导入存档」按钮，选择 `.dat` 存档文件
3. 编辑所需的属性值
4. 点击「导出存档」按钮保存修改后的存档
5. 将导出的文件覆盖原存档位置

### 3. 功能说明

#### 基础属性
- 金钱、压力、行动力等基础资源
- 善恶平衡值
- 学院职业选择

#### 详细属性
- 四维属性（体力/智力/魅力/感性）
- 16项子属性
- 战斗等级和经验

#### 装备
- 服装、武器、防具ID

#### NPC
- 37个NPC的好感度、见面次数等

#### 物品/技能
- 物品持有数量
- 技能解锁状态

#### 轮回数据
- 玩家/父亲名称
- 生日设置
- 恋爱状态
- 成就进度

#### 快捷操作
- 一键满属性
- 金钱最大化
- 压力清零
- 全NPC满好感
- 解锁全部技能

## 开发

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建单文件版本
```bash
npm run build
```

构建结果在 `dist/index.html`

## 技术栈

- Vue 3
- Vite
- CryptoJS (AES加密)
- vite-plugin-singlefile (单文件打包)

## 注意事项

- 存档使用AES加密，密钥已从游戏代码中提取
- 修改存档前请务必备份原文件
- 不合理的数值可能导致游戏异常
- 建议在游戏关闭时修改存档
